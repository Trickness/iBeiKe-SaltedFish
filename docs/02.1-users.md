# users.php       
users.php 提供用户基本操作  

```
用户信息:
    - 学号
    - 姓名
    - 昵称
    - 班级
    - 宿舍号
    - 手机号
```

## 用户信息结构（JSONStr）
```
{
    student_id : {
        access  : "ACCESS",
        value   : "STRING"
    },

    name       : {
        access  : "ACCESS",
        value   : "STRING"
    },

    nickname   : "STRING",
    header     : "URL",

    class_info  : {
        access  : "ACCESS",
        department : {
            access  : "ACCESS",
            value   : "STRING"
        },
        enrollment : {  // 入学年份
            access  : "ACCESS",
            value   : "INT"
        },
        class_no     : {
            access  : "ACCESS",
            value   : "INT"
        }
    },

    dormitory  : {
        access  : "ACCESS",
        dormitory_id : {
            access    : "ACCESS",
            value     : "INT"
        },
        room_no      : {
            access    : "ACCESS",
            value     : "INT"
        }
    },

    phone_number : {
        access    : "ACCESS",
        value     : "STRING"
    }
}
```

注：    
```
ACCESS :
    - public        公开可见
    - protected     部分可见(暂时不用)
    - privated      他人完全不可见（暂时不用）
```

### 公开API
```php
/**
 *
 * 将论坛账号与学生信息绑定在一起：
 *      - 通过利用学生账号密码登陆本科教学网，获得基本信息来绑定
 *      - 验证成功即绑定
 *
 * @param
 *      - (@String) original_un     源论坛用户名 
 *      - (@String) original_pw     源论坛密码
 *      - (@String) student_id      学号
 *      - (@String) student_pw      学号绑定的密码
 *
 * @return
 *      - (@String) session_key     会话密钥，作为后面所有操作的唯一凭证
 *
 **/
user_bind($original_un,$original_pw,$students_id,$student_pw);



/**
 * 
 * 创建用户
 * 
 * 
 * 
 * 
 * 
 * 
 **/
user_create($student_id,$password);


/**
 * 
 * 登陆到咸鱼站
 *
 * @param
 *      - 账号和密码（待定)
 * 
 * @return
 *      - (@String) session_key 是咸鱼站操作的唯一 key  用于用户的各种操作时的认证
 *                                  十天后自动失效
 *
 **/
user_login($username,$password);



/**
 * 
 * 用户登出
 * 
 * @param
 *      - (@String) session_key
 * 
 **/
user_logout($session_key);


/**
 * 
 * 获得自己的信息
 * 
 * @param
 *      - (@String) session_key
 *
 * @return
 *      - (@JSONStr) user_info
 * 
 **/
fetch_self_info($session_key)


/**
 * 
 * 获得他人的信息
 * 
 * @param
 *      - (@String) session_key
 *      - (@String) student_id
 *
 * @return
 *      - (@JSONStr) user_info
 * 
 **/
fetch_user_info($session_id,$student_id)



/**
 * 
 * 更新自己的信息
 * 
 * @param
 *      - (@String) session_key
 * 
 * @return
 *      - (@JSONStr) updated_user_info
 *
 **/
update_self_info($session_key)
```
